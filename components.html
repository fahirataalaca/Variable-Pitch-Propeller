Key Components
Component Description
Stepper Motor (NEMA 11) The primary actuator for the propeller's pitch adjustment.
DRV8825 Driver Microstepping driver used to control the stepper motor with microsteps.
Li-ion Batteries Compact and rechargeable power source for the motor and electronics.
XIAO ESP32 C3 The microcontroller responsible for controlling the stepper motor.
Custom PCB The controller board responsible for controlling the stepper motor and receiving data from the hall sensor. Used in the actual tests with the propeller.
3D Printed Propeller The propeller blades and housing designed using Fusion 360 and printed with PLA+ for durability.
Brushed DC Motor The motor responsible for spinning the propeller as a whole.
Waveshare 5" HDMI Screen Displays the RPM and theoretical thrust data in real time.
L298N DC Motor Driver DC motor driver used to control the brushed DC motor.
Raspberry Pi 4 Microcontroller responsible for controlling the DC motor driver, displaying data on the screen, and receiving data from the ESP32 C3.
Hall Effect Sensor Sensor responsible for propeller RPM detection.
Bearings The 608ZZ bearings inside the propeller are crucial for smooth operation. The pillow block bearings take the load off the DC motor and ensure the propeller spins smoothly.
Additional Components

Aluminium profiles, Raspberry Pi 4 protection and cooling box, square 3-axis profile connectors, wires.
